# Fask (실시간 협업 워크스페이스 및 칸반 관리 플랫폼)

> AI 대화 내용 분석을 통한 요약본 및 칸반 보드 자동 생성을 목표로 하는 <br>실시간 협업 채팅 서비스입니다.

## 🛠 Tech Stack
- **Frontend:** React, Vite
- **Backend:** Node.js, Express, Socket.io
- **Database:** MySQL
- **Collaboration & CI/CD:** Git, GitHub
